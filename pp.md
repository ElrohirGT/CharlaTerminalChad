# ¿Qué es una Shell?

![](./imgs/bashIcon.png)

<!-- pause -->

Conceptualmente es una interfaz textual en la cual ingresamos comandos dentro de la computadora y nos devuelve un resultado.

<!-- column_layout: [1,1] -->

<!-- column: 0 -->
```bash +exec
# Single line command
echo "Hello World!"
```

<!-- column: 1 -->
```bash +exec
echo "Hello" \
    "World!"
```
